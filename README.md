# Laptop

Set environment variable:

```
export LAPTOP="$HOME/laptop"
```

Clone:

```
gh repo clone croaky/laptop $LAPTOP
cd $LAPTOP
```

Review:

```
less laptop.sh
```

Run:

```
./laptop.sh
```
